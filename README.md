1. Установка зависимостей
npm install

2. Запуск сервера
npx node-ts server.ts

3. Запуск фронтенда
npm run dev

Документы сохраняются в локальный JSON-файл.

3. Реализовать редактирование документов
4. Добавить поиск и фильтрацию
